Service

Upload a demo archive. Get back a product zip.

The service accepts compressed demo projects, runs Demo2Project's multi-agent harness, then returns a normalized product package as a zip file.

Input: zip, 7z, rar, tar, tar.gz, tgz Output: zip

No archive selected.

  1. 1. Intake Archive type, entrypoints, dependencies and risk boundaries are detected before any agent edits files.
  2. 2. Iterate Analyzer, Planner, Executor, Verifier, Reviewer and QA Memory run repeated demo-to-product loops.
  3. 3. Return The completed artifact is packaged as a product zip with reports, docs, tests and verification evidence.
