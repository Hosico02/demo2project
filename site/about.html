About

A harness that makes demo-to-product work measurable.

Demo2Project is not a single coding agent. It is a control layer above agents, built to decide whether a demo has actually become a maintainable product and to keep pushing until the evidence says yes.

Current State

The project already includes project scoring, gap analysis, deterministic rule-based execution, MiniMax provider support, verification gates, QA memory, regression learning, long-horizon autonomy and archetype-specific harnesses for UI, CLI, API, config, data and workers.

Future State

The target is an industrial multi-agent productization platform: users upload a demo archive, the system runs long-horizon supervised iteration, validates every claim with evidence and returns a product-ready zip artifact.
